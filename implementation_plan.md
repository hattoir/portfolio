# 実装計画: ハードウェア・ロボットエンジニア向けポートフォリオサイト

ハードウェアやロボット開発の経歴に合わせた、近未来的で洗練されたデザイン（ダークモード、ネオンカラーのアクセント、スムースなアニメーション）のポートフォリオサイトを作成します。

## 目標
ロボットやハードウェアの雰囲気が伝わるデザインと、こだわりのアニメーションを取り入れたWebサイトを構築します。

## ユーザーへの確認事項
* フレームワークの指定は特にありませんでしたので、読み込みが速くアニメーションの制御がしやすい **HTML + CSS (Vanilla) + JavaScript** の構成で進めます。Reactなどのフレームワークを使用したい場合はお知らせください。
* 全体的なカラーテーマは「ダークグレー＆サイバーブルー（またはオレンジ等）」を想定しています。お好みのアクセントカラーがあれば教えてください。

## 提案するファイル構成と変更内容
開発は `c:\portfolio` フォルダ内で行います。

### コンポーネントおよびページ構成
* `index.html` (ホーム画面。ハードウェアを連想させるヒーローセクション)
* `works.html` (作品ページ。後日の詳細条件に合わせて枠組みだけ作成します)
* `activities.html` (これまでの活動実績。月ごとのタイムライン形式で、クリックするとモーダルまたは別画面で「イベント詳細」「写真」「学んだこと」を表示する仕組みを作ります)
* `learning.html` (読んだ技術書・PDF・サイトをまとめたページ。リンク、学んだ内容、感想をカード形式等で表示します)

### アセット構成
* `assets/css/style.css` (全体デザイン。CSS変数を用いたテーマ管理と、ハードウェア的な幾何学・発光アニメーションを実装)
* `assets/js/main.js` (ページ遷移時のアニメーション、モーダルの制御、スクロールに応じた要素のフェードインなどを実装)
* `assets/images/` (写真などのアセット配置用フォルダ)

## 検証計画
### 手動検証
1. ブラウザで `index.html` 等の各ページを開き、サイバー/ハードウェア風のデザインが意図通りに表示されているか確認します。
2. ナビゲーションやホバー時のアニメーション、スクロール時の動作がスムーズに機能しているか手動でテストします。
3. `activities.html` の月別表示から詳細を開く動作、`learning.html` のリンククリック動作が正しく機能するか確認します。
